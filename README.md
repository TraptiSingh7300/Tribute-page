# Tribute Page

A simple responsive tribute page built with HTML and CSS.

Live Demo

https://traptisingh7300.github.io/Tribute-page/

About

This repository contains a small static project created to practice building semantic HTML and modern CSS. The page includes a header, tribute content, and responsive layout.

How to view locally

1. Clone the repository:
   ```bash
   git clone https://github.com/TraptiSingh7300/Tribute-page.git
   ```
2. Open `index.html` in your browser.
